## 📂 Featured Projects

### [Heart-Disease-Prediction-Using-Machine-Learning](https://github.com/TranVietThang178/Heart-Disease-Prediction-Using-Machine-Learning)
Comparing Logistic Regression, Random Forest, and XGBoost on the Cleveland Heart Disease Dataset:
- **Machine Learning:** Classification models with cross-validation
- **EDA & Feature Engineering:** Handling missing values, encoding, scaling
- **Best Result:** XGBoost — 87% accuracy, AUC-ROC 0.93

### [Pneumonia-Detection-Using-Deep-Learning](https://github.com/TranVietThang178/Pneumonia-Detection-Using-Deep-Learning)
CNN-based binary classifier for chest X-ray pneumonia detection:
- **Deep Learning:** Custom CNN architecture with transfer learning
- **Data Augmentation:** Robust generalisation on medical imaging data
- **Stack:** Python, PyTorch, OpenCV

### [Global-Life-Expectancy-Analysis](https://github.com/TranVietThang178/Global-Life-Expectancy-Analysis)
End-to-end ELT pipeline analysing WHO life expectancy data (193 countries, 2000–2015):
- **ELT Pipeline:** Data ingestion, cleaning, and transformation
- **Analysis:** SQL queries, statistical analysis, trend identification
- **Visualisation:** Tableau dashboards and Matplotlib/Seaborn charts

### [AI-Annotation-Tool](https://github.com/TranVietThang178/AI-Annotation-Tool)
Full-stack AI-powered video annotation platform — Published IGI Global book chapter (DOI: 10.4018/979-8-3693-9728-8.ch009):
